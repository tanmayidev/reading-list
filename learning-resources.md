## λ Functional Programming <a id="functional-programming"></a>

> **Recommended learning path for Node.js developers:**
> 1. Start with *Grokking Simplicity* — practical and beginner-friendly, no math required
> 2. Work through *Professor Frisby's* egghead course for hands-on JS FP
> 3. Read *Mostly Adequate Guide* (free) to deepen understanding
> 4. Then pick up Elm → Elixir → OCaml to explore typed/pure FP

### JavaScript Functional Programming <a id="js-functional"></a>

**Books**

- [Grokking Simplicity — Eric Normand](https://www.manning.com/books/grokking-simplicity) 📖 *(paid — best beginner FP book for JS/Node devs)* ⭐
- [Mostly Adequate Guide to FP](https://mostly-adequate.gitbook.io/mostly-adequate-guide/) *(free)*
- [Functional Light JavaScript — Kyle Simpson](https://aguru.gitbooks.io/functional-light-javascript/content/) *(free)*

**Courses**

- [Professor Frisby Introduces Composable Functional JavaScript — egghead.io](https://egghead.io/courses/professor-frisby-introduces-composable-functional-javascript) *(free course)* ⭐
- [Functional JavaScript Learning Path — Frontend Masters](https://frontendmasters.com/learn/functional-javascript/) *(paid — requires monthly subscription)*

**Blogs & Articles**

- [James Sinclair's Blog — jrsinclair.com](https://jrsinclair.com/) ⭐ excellent in-depth FP in JS articles
- [Eric Normand's Blog — ericnormand.me](https://ericnormand.me/)

**Key Concepts to Learn**

- Pure functions & side effects
- Immutability
- Function composition & currying
- Functors, Monads, Applicatives
- Higher-order functions (map, filter, reduce)
- Point-free style

## 🏗️ Object Oriented Programming <a id="oop"></a>

> Even as a functional programmer, understanding OOP is essential for reading most codebases and for system design.

**Books**

- [Head First Design Patterns](https://www.oreilly.com/library/view/head-first-design/9781492077992/) *(paid — most beginner friendly)*
- [Clean Code — Robert C. Martin](https://www.oreilly.com/library/view/clean-code-a/9780136083238/) *(paid)*
- [Object-Oriented Thought Process](https://www.oreilly.com/library/view/the-object-oriented-thought/9780135182130/) *(paid)*

**Free Resources**

- [OOP in JavaScript — javascript.info](https://javascript.info/object-oriented-programming)
- [OOP Concepts — MDN](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/Object-oriented_programming)
- [Software Design Architecture Roadmap](https://roadmap.sh/software-design-architecture)

**Core Concepts to Learn**

- The four pillars: Encapsulation, Abstraction, Inheritance, Polymorphism
- SOLID Principles
- Composition vs Inheritance
- Interfaces & Abstract Classes
